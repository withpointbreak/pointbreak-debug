# Contributing to Pointbreak Debug

Pointbreak Debug is the legacy debugger-bridge product. Its final VS Code Marketplace release is
`0.2.5`, and its private implementation is not accepting code contributions.

You can still help maintain the historical product surface:

- search [existing issues](https://github.com/withpointbreak/pointbreak-debug/issues) before filing a
  report;
- use the [bug report](https://github.com/withpointbreak/pointbreak-debug/issues/new?template=bug_report.yml)
  for reproducible defects in the Debug extension, CLI, or bridge;
- use the [question template](https://github.com/withpointbreak/pointbreak-debug/issues/new?template=question.yml)
  or [Discussions](https://github.com/withpointbreak/pointbreak-debug/discussions) for usage help;
- identify the affected Pointbreak Debug version and include relevant editor, operating-system, and
  diagnostic details.

Do not include credentials, private source code, or other sensitive information in a public issue.
Do not report vulnerabilities publicly; follow [SECURITY.md](SECURITY.md) instead.

The new Pointbreak product is a separate, incompatible project. Requests for that product should not
be filed against this Debug repository merely because the extension keeps the same Marketplace
identity.
